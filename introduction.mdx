<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-gray-600 dark:text-gray-400 mb-8">Get ShelfWise up and running in minutes</p>
  
  <Steps>
    <Step title="Clone and install dependencies">
      Clone the repository and install PHP and Node.js dependencies.
      
      ```bash
      git clone https://github.com/gHoSTeCHs/shelfwiser.git
      cd shelfwiser
      composer install
      npm install
      ```
    </Step>
    
    <Step title="Configure environment">
      Copy the example environment file and generate your application key.
      
      ```bash
      cp .env.example .env
      php artisan key:generate
      ```
      
      <Note>Update your `.env` file with database credentials and other configuration values.</Note>
    </Step>
    
    <Step title="Run migrations">
      Set up the database schema and seed initial data.
      
      ```bash
      php artisan migrate --seed
      ```
    </Step>
    
    <Step title="Start the development server">
      Launch the application with Laravel server, queue worker, and Vite for asset compilation.
      
      ```bash
      composer dev
      ```
      
      <Info>This runs `php artisan serve`, `php artisan queue:listen`, and `npm run dev` concurrently.</Info>
    </Step>
  </Steps>
</div>
